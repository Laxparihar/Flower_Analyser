Flower Analyser Model

Welcome to the Flower Analyser Model

This API is designed for flower analysis. You must send both the file type and the flower image file to use this API.

The API processes flower images in formats such as PNG, JPG, etc. Ensure that the correct file type is included in your request.

How to Use the API

To use the Flower Analyser API, follow these steps:

  • Step 1: Upload the flower image file in either PNG or JPG format.
  • Step 2: Specify the file type in your request body.
  • Step 3: Receive the prediction results including flower type, color, and estimated oil concentrations.

Sample Prediction Output:

    {
        "predicted_flower_type": "Rose",
        "predicted_flower_color": "Red",
        "estimated_oil_concentrations": {
            "Linalool": 0.23,
            "Geraniol": 0.15,
            "Citronellol": 0.30
        }
    }
